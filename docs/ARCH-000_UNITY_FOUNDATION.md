# IL JOURNEY

Il Journey rappresenta lo stato di crescita del collaboratore.

Ogni persona si trova sempre in un solo punto del Journey.

Il Journey determina:

- Dashboard
- Missioni
- Suggerimenti
- Priorità
- Workflow
- Notifiche

---

# LE MISSIONI

Le Missioni rappresentano gli obiettivi.

Ogni Missione è composta da Azioni.

Una Missione termina solo quando tutte le Azioni richieste sono completate.

---

# LE AZIONI

- chiara

- verificabile

# I COMPLETAMENTI
